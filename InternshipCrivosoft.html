Introduction to the internship

This internship was carried out within the scope of the curricular unit Internship, in the Professional Higher Technical Course (CTeSP) in Information Systems Technologies and Programming.
During the internship, the host entity was Crivosoft. The company is considered a digital marketing micro-enterprise and is in Santarém.
The internship lasted 751 hours spread over 95 days.
In this internship I was part of a team of 2 elements as a full stack developer.




The Main Project

In my internship at crivosoft I worked in a project that consisted of the development of a system to promote and generate leads for healthcare providers.
Consisting of a:
• Front office as an Android app using Java with Android Studio.
• BackOffice with as a website using html, CSS, and bootstrap.
• Database in SQL with SQL Server.
• REST APIs using C# with .NET Framework and Entity Framework.

The main activity I had during this project was the development of the APIs and the android app

Technologies: SQL, HTML5, JavaScript, Java SE, C#

Tools: SQL Server 2012, SQL Server Management Studio, Android Studio, Visual Studio 2019

Others: MVC, REST API, Entity Framework, .Net Framework
