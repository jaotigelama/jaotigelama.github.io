Be There Or Be Square

3D platformer

The game

This game is a simple platformer where the player must reach the end of the level before the timer runs out and without falling, it has 3 levels, and they can be selected from the menu. There are 3 different sizes of platforms and 3 different behaviours, it uses a 3rd person camera with look around capabilities.

The Project

This project was a 3-day project where I was the only developer, this project had the objective of solidifying my knowledge in unity very early in the learning of the engine and it has my first original game in unity.
